# aeo-audit

> Initial scaffold. Full package contents incoming in the next push.

The open AEO/GEO audit standard. MIT-licensed. CLI + library for scoring any URL against the 47-signal VectorCite Rubric.

Run on any URL:

```bash
npx aeo-audit https://example.com
```

See [vectorcite.digiocular.com/rubric](https://vectorcite.digiocular.com/rubric) for the full methodology.
